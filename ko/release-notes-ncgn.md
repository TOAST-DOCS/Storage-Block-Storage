## Storage > Block Storage > 릴리스 노트

<a id="june-2-2026"></a>
## 2026. 06. 02.

### 기능 개선/변경

* 블록 스토리지 연결 정보 표기 개선
    * 종료된 인스턴스에 연결된 블록 스토리지도 어느 인스턴스에 연결되어 있는지 확인할 수 있습니다.
    * 기존 **연결 정보** 열이 **연결 인스턴스** 열로 변경되었습니다.
* 블록 스토리지 및 스냅숏 목록 조회 API의 limit 파라미터 기본값(default)이 100개, 최댓값(max)이 1,000개로 조정됩니다.
* 블록 스토리지 **연결 관리** 버튼 분리
    * 기존 **연결 관리** 버튼이 **연결 추가**와 **연결 해제** 버튼으로 분리됩니다.

<a id="may-29-2025"></a>
## 2025. 05. 29.

### 기능 추가

* 블록 스토리지 연결 시 삭제 정책 설정 기능이 추가됩니다.

<a id="march-6-2025"></a>
## 2025. 03. 06.

### 기능 추가

* 블록 스토리지당 최대 스냅숏 수를 제한하는 쿼터가 추가됩니다.

<a id="june-4-2024"></a>
## 2024. 06. 04.

### 기능 추가

* 블록 스토리지 크기 변경 기능이 추가됩니다.

<a id="august-29-2023"></a>
## 2023. 08. 29.

### 기능 개선/변경

* 용어 변경
    * 볼륨, 디스크 등의 용어를 **블록 스토리지**로 변경했습니다.

<a id="march-28-2023"></a>
## 2023. 03. 28.

### 기능 개선/변경

* API 엔드포인트가 변경됩니다.

<a id="october-4-2022"></a>
## 2022. 10. 04.

### 기능 개선/변경

* **스냅숏에서 블록 스토리지 생성** 기능 수행 시 스냅숏 무결성 확인 절차가 추가됩니다.

<a id="june-16-2020"></a>
## 2020. 06. 16.

### 기능 추가

* Public API v2 출시
    * Public API v2는 Openstack API와 호환됩니다.

### 기능 개선/변경

* 생성할 수 있는 블록 스토리지의 최대 크기가 1TB에서 2TB로 증가합니다.

<a id="november-26-2019"></a>
## 2019. 11. 26.

### 기능 개선/변경

* 기본 디스크가 블록 스토리지 목록에 노출됩니다.

<a id="january-29-2019"></a>
## 2019. 01. 29.

### 기능 개선/변경

* 블록 스토리지 생성 시 원본으로 스냅숏을 선택할 수 없도록 변경됩니다.
* 스냅숏으로 블록 스토리지 생성 시 가용성 영역과 블록 타입을 선택할 수 있도록 변경됩니다.

<a id="september-21-2017"></a>
## 2017. 09. 21.

### 기능 추가

* Public API 추가
    * Object Storage에 이어 Block Storage를 API로 사용할 수 있습니다.
    * 현재 제한적인 기능만 이용할 수 있으며, 추후 API가 추가되어 기능이 확장될 예정입니다.
    * [블록 스토리지 API 가이드](/Storage/Block%20Storage/ko/public-api/)를 참고하세요.

<a id="july-20-2017"></a>
## 2017. 07. 20.

### 버그 수정

* 블록 스토리지 생성 시 간헐적으로 생성이 완료되지 않던 버그가 수정되었습니다.

<a id="january-19-2017"></a>
## 2017. 01. 19.

### 기능 개선/변경

* 블록 스토리지 연결 관리 기능에 동일 가용성 영역의 인스턴스에만 연결이 가능함을 명시했습니다.

### 버그 수정

* 스냅숏으로 블록 스토리지 생성 시 간헐적으로 실패하는 문제가 수정되었습니다.
